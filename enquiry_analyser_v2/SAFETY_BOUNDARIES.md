# Meridian Enquiry Analyser V2 Safety Boundaries

This V2 artifact uses fictional data only. It is not production software and is not intended for autonomous customer-facing use.

## Boundaries

- No diagnosis from enquiry text.
- No treatment decisions from enquiry text.
- No guaranteed PRP, SMP, hair unit, AKN, or hair restoration outcomes.
- No customer-facing reply should be sent without owner approval.
- Human review is required for sensitive clinic, treatment, diagnosis, suitability, outcome-guarantee, prompt-injection-like, missing-guidance, or low-confidence retrieval cases.
- The analyser can organize details, retrieve local guidance, flag risks, and recommend a next step.

## Data Boundary

- Uses fictional examples only.
- Does not use customer records.
- Does not call outside services.
- Does not contain clinical validation.
